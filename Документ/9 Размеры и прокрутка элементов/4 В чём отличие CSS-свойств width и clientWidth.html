В чём отличие между getComputedStyle(elem).width и elem.clientWidth? 1.для инлайновых элементов значение width, полученное посредствам getComputedStyle(elem).width может быть равно auto, в то время как clientWidth возвращает всегда конкретное значение 2.свойство clientWidth возвращает нам реальную ширину содержимого, в то время как с width иначе - из-за появления прокрутки значение width может меняться в зависимости от браузера. 3.clienWidth возвращает конкретное значение-число, а getComputedStyle(elem).width это строка с px
